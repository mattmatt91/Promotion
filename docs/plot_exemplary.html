plot_exemplary module

This module plots exemplary measurements. A plot is created for each sensor. For these, a file must be created manually for each sensor in which the corresponding samples (column name must correspond to the name of the sample) are entered. In addition, a time axis should be available. The file name must be created as follows: sensorname _compare.csv These measurements must be stored in a folder called results//exemplary.
